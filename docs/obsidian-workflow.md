# Obsidian Workflow

Keep the Obsidian repository private. Publish only cleaned markdown copies here.

## Add a Post

1. Copy a note into `content/posts`.
2. Remove private context and unfinished backlinks.
3. Add frontmatter with title, date, and tags.
4. Add the post to `content/posts-index.json`.

```md
---
title: Example Post
date: 2026-05-27
tags:
  - web
---

# Example Post

Write the public version here.
```
