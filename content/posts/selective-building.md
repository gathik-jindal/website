# Selective Building in Public

Building in public works best when it is edited.

Capture everything privately, then publish only the note that helps someone understand the project, decision, or lesson.

## Lightweight rule

- Draft in Obsidian.
- Mark good candidates with `public`.
- Copy a cleaned version into `content/posts`.
- Add it to `content/posts-index.json`.
