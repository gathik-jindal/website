# Regex

Regular expression, or regex, is a sequence of characters that describes a set of strings that match a pattern. This is very useful in programming because often we
want to match our input with something that our program is expecting.

Mostly all programs use a similar regex syntax and its very simple to learn, just writing this so we know something like the exists.
